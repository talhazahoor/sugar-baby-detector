# Sugar Baby Detector
## Description
A mobile app built with React Native, TypeScript, Expo, Firebase, and Supabase to detect and prevent sugar baby relationships.
## Prerequisites
* Node.js
* Expo
* Firebase
* Supabase
## Installation
* npm install
## Environment setup
* cp .env.example .env
* Fill in the environment variables
## Run commands
* npm start (for development)
* npm run build (for production)
